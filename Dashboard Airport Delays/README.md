# Airport Delays Dashboard in Power BI (FP20 Challenge 23)

In this tutorial, you’ll learn how to build a simple and effective Power BI dashboard to analyze airline delays using a real dataset.

This is part 2 of the FP20 Challenge series, focusing on turning data into clear and actionable insights.

---

## 🎥 Watch the tutorial

[FP20 Challenge 23 – Airline Delay Dashboard](https://youtube.com/watch?v=0iUDQ6dO4Ug&feature=youtu.be)

---

## 🧠 What this project does

This dashboard helps you analyze airline delay performance across carriers and routes.

It allows you to:
- compare airline performance  
- identify delay patterns  
- analyze key metrics across carriers  
- build a structured overview of delays  
- turn raw data into meaningful insights  

---

## 🚀 What you’ll learn

In this tutorial, you’ll see:

- how to build a beginner-friendly dashboard  
- how to structure insights clearly  
- how to visualize airline performance  
- how to work with a real dataset  
- how to design a clean and intuitive report
